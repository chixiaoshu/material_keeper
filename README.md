# material_keeper

## !!!目前仅属于chixiaoshu学习rust以及自行使用。

一个用于管理物料信息的 Rust 项目，支持添加、查询和更新物料数据。

## 项目结构

- `material_core/`: 核心功能模块
- `material_lib/`: 公共库，定义了 `Material` 结构体等
- `material_gui/`: 图形用户界面模块（待开发）
- `material_scan/`: 扫描功能模块（待开发）
